# WEB103 Project 1 - *Sportify*

Submitted by: **Danh Nguyen**

About this web app: **Sportify is a lightweight web app for discovering upcoming sports events at a glance. Browse a clean grid of equal-height cards with images, matchup, date/time, venue, and price tier, then tap “Read More” for a simple detail view. Built with Vite + vanilla JS and an Express API, it’s fast, responsive, and easy to extend.**

Time spent: **4** hours
